Sambaインストール

Tue Dec 26 2017

事前準備

ローカルからサーバーにSSH接続できる


SELinuxを無効化

SELINUXを無効化

$ vi /etc/selinux/config
SELINUX=disabled

そして反映！

$ setenforce 0
setenforce: SELinux is disabled

iptablesの設定

ローカルだしアクセス制限なくしちゃいます。

$ vi /etc/sysconfig/iptables

自分は下記2つをとりあえずコメントアウト

#-A INPUT -j REJECT --reject-with icmp-host-prohibited
#-A FORWARD -j REJECT --reject-with icmp-host-prohibited

変更後、再起動！

$ /etc/init.d/iptables restart

sambaのインストール・設定

sambaをインストール

$ yum -y install samba

sambaを設定

$ vi /etc/samba/smb.conf

変更は以下の通り

[global]
unix charset = UTF-8 #追記
dos charset = CP932 #追記
security = share #userから変更
load printers = no #yesから変更
cups options = raw
guest account = root #追記
disable spoolss = yes #追記

#以下追加
[root]
comment = All
path = /
browseable = yes
writable = yes
guest ok = yes
guest only = yes
create mode = 0777
directory mode = 0777
share modes = yes

これでsambaの設定終了。 起動させましょう！

$ /etc/init.d/smb start

常に実行するように設定

$ chkconfig smb on

これでファイルの接続が可能になりました!


接続方法

Windows

スタートからプログラムとファイルの検索に「\サーバーIPアドレス」 

Mac

「Finder」からサーバへ接続機能を使って、ローカルのIP打ち込む
